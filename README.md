
# Create: Chain Conveyors Compat
This mod for NeoForge 1.21.1 allows chains from any mod to work with Create Chain Conveyors.

This mod was initially intended for only TerraFirmaCraft chain compatibility, but has since expanded.
The code is partially based on the implementation of this feature in the [WoodenCog](https://www.curseforge.com/minecraft/mc-mods/woodencog) mod for Forge 1.20.1 by [Manwe_b](https://www.curseforge.com/members/manwe_b).
